# Визначення змінних

Визначити субʼєкт виконання, обʼєкт якого можна перезаписати, можна використовуючи слово `змінна`:

```ціль
змінна число_ПІ: д64;

надрукувати_д64(число_ПІ); // якесь брудне значення

число_ПІ = 3.14;

надрукувати_д64(число_ПІ); // 3.14
```

Для зручності, під час визначення можна відразу вказати початкове значення:

```ціль
змінна число_ПІ: д64 = 3.14;

надрукувати_д64(число_ПІ); // 3.14

число_ПІ = 3.15;

надрукувати_д64(число_ПІ); // 3.15
```
