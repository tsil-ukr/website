# Створення дій

Створити та визначити дію можна використовуючи слово `дія`:

```ціль
дія додати(а: дійсне, б: дійсне): дійсне {
  вернути а + б;
}
```

Дію можна визначити без тіла. Створити тіло можна пізніше:

```ціль
дія додати(а: дійсне, b: дійсне): дійсне;

// ...

дія додати(а: дійсне, b: дійсне): дійсне {
  вернути а + b;
}
```
