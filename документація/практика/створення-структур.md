# Створення структур

Створити та визначити структуру можна використовуючи слово `структура`:

```ціль
структура Точка {
  ікс: десяткове;
  ігр: десяткове;
}
```

Структуру можна визначити без тіла. Створити тіло можна пізніше:

```ціль
структура Точка;

// ...

структура Точка {
  ікс: десяткове;
  ігр: десяткове;
}
```
