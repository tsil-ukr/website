# Структури

Запис структур в Цілі відбувається так:

```ціль
структура Людина; // декларація
```

```ціль
структура Людина {
  імʼя: кд;
}
```

```ціль
структура Машина<Д> {
  двигун: Д;
}
```

За замовчуванням структури мають вирівнювання.
Щоб його прибрати, можна додати атрибут `\пакована`:

```ціль
\пакована структура Дані {
  а: ц32;
  б: логічне;
  в: кд;
}
```

Ця поведінка ідентична до _C (eng)_ з `__attribute__((packed))`.

> Назва атрибуту може змінитись в майбутньому.
